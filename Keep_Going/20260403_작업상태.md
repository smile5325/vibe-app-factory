## [버그 수정] AI 주제 생성 버튼 3연속 에러 수정

### Bug 1 — 버튼 무반응
- 원인: `generateTopics()` 내 모든 에러를 `console.error` + `return`으로 묵살
- 수정: `topicError` state 추가, 에러 발생 시 빨간 박스로 UI에 표시

### Bug 2 — JSON 파싱 크래시
- 원인: `await res.json()` 직접 호출 → 빈 응답/에러 응답 시 예외 발생
- 수정: `res.text()` 먼저 받은 후 `JSON.parse()` 수동 호출 (6단계 방어 파싱)

### Bug 3 — /api/claude 404
- 원인: Vite 개발 서버는 `api/claude.js` (Vercel 서버리스) 파일을 인식 못함
- 수정: `vite.config.js`에 `localApiPlugin()` 인라인 미들웨어 추가
  → `/api/claude` 요청을 Anthropic API로 직접 프록시

### Bug 4 — ANTHROPIC_API_KEY undefined
- 원인: Vite 플러그인 컨텍스트에서 `process.env`는 `.env` 파일을 자동 반영 안 함
- 수정: `defineConfig(({ mode }) => {...})` 팩토리 패턴 전환
  + `loadEnv(mode, process.cwd(), "")` 사용 (prefix="" = 모든 키 로드)
  + `env.ANTHROPIC_API_KEY` 로 읽기
